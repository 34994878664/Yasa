# 萌渊地牢 v8 可爱角色 + 核心BD + Boss动作版

这版重点更新：

- 主角重画成圆头小豆丁，不再使用上一版怪诞兔耳形象。
- 加强妈刀系、硫磺火系、科技X系，拿到后就是通关级核心BD。
- 加入猫猫套装：3件套激活友方苍蝇，苍蝇吃玩家伤害和射速加成。
- Boss动作增强：冲刺、踩踏、落石、召唤、十字激光、炸弹雨，不只会发弹幕。
- 继续保留：BD按钮、BD图鉴、属性面板、天使/恶魔房、BGM开关、透明摇杆、竖屏进门按钮。

## 上传方法

解压 zip 后，把里面所有文件上传到 GitHub 仓库最外层，覆盖旧文件，然后 Commit changes。

如果手机浏览器还是旧版，在网址后加：

```text
?v=8
```

例如：

```text
https://你的用户名.github.io/仓库名/?v=8
```
