# 萌渊地牢 Cute Abyss

这是一个可以直接上传 GitHub Pages 的原创网页地牢肉鸽射击游戏。

## 文件说明

- `index.html`：网页入口，GitHub Pages 会自动打开它
- `styles.css`：手机适配和UI样式
- `game.js`：完整游戏逻辑
- `manifest.webmanifest` / `icon.svg`：手机添加到桌面的图标配置
- `.nojekyll`：让 GitHub Pages 原样发布文件
- `robots.txt` / `sitemap.xml`：给搜索引擎用，可以后面把 sitemap 里的网址改成你的 GitHub Pages 网址
- `上传教程_手机_GitHub.md`：手机上传 GitHub 仓库并生成网址教程

## 当前玩法核心

- 房间推进：普通房、宝物房、商店、诅咒房、Boss房
- 手机操作：左摇杆移动，右摇杆手动射击，默认自动瞄准
- 电脑操作：WASD移动，方向键/鼠标射击，空格暂停
- 生命机制：红心、魂心、黑心
- 特殊房：Boss后概率开启天使房或恶魔房
- 无红伤：整层没有红心伤害，特殊房概率大幅上升；魂心/黑心挡伤不破无红伤
- 天使房：免费强力奖励，偏防御、魂心、复活、圣光、无伤成长
- 恶魔房：用红心上限换爆发，偏飞刃、光束、环形激光、多重、低血狂暴
- BD数量：内置 100+ 道具，包含射击形态改变、状态流、跟班、环绕物、高血量增伤、低血狂暴、金币/钥匙成长等
