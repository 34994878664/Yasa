# 萌渊地牢 Cute Abyss

这是一个可直接上传到 GitHub Pages 的原创网页小游戏。

## 重要说明

- 打开游戏后，**出生房是安全房，所以一开始没有怪物是正常的**。
- 点击「开始新局」后，往右边金色门走，进入第一战斗房就会刷怪。
- 手机操作：左摇杆移动，默认自动射击；右摇杆可以手动控制射击方向。
- 核心机制：保护红心。魂心/黑心挡伤不破无红伤，Boss 后更容易开强力天使/恶魔房。

## 上传 GitHub Pages

把本文件夹里的所有文件上传到仓库最外层，不要只上传 zip。
必须包含：

- index.html
- styles.css
- game.js
- manifest.webmanifest
- icon.svg
- robots.txt
- sitemap.xml
- .nojekyll

然后在 GitHub：Settings → Pages → Deploy from a branch → main → /root → Save。
