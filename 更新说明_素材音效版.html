萌渊地牢：素材音效版更新说明

这版重点不是改玩法，而是改观感和听感：角色、小怪、Boss、拾取物、地牢砖块、命中特效、拾取音效、进门音效和 BGM 开关。

这版改了什么

  • 角色、小怪、Boss 换成更可爱的 16×16 像素素材包风格。
  • 地牢地砖、墙、障碍物、尖刺、金币、钥匙、炸弹、红心、魂心、黑心都换了视觉。
  • 飞刃、眼泪、追踪弹、穿透弹、环形激光、光束都有更明显的视觉反馈。
  • 加入打击音效、击杀音效、拾取音效、进房音效、Boss提示音。
  • 右上角新增 BGM关 / BGM开 按钮，玩家自己决定开不开。

上传方式

直接把这一版解压后的所有文件上传到 GitHub 仓库最外层即可。

重点确认仓库最外层有：index.html、game.js、styles.css、.nojekyll。

这次有没有额外素材文件夹？

不用担心。像素精灵已经嵌入到 game.js 里了，所以手机上传时不需要单独上传 assets 文件夹。你还是“一股脑上传文件”就行。

刷新还是旧版怎么办？

手机浏览器可能缓存旧版。把网址后面加：

?v=3

例如：你的网址/?v=3
